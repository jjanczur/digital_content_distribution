## Deployment

```shell
npm install -g truffle
npm install -g ganache-cli
npm install
ganache-cli # Start a blockchian
truffle compile
truffle migrate
truffle test
```

## Linting

Lint all smart contracts:

```shell
solhint "contracts/**/*.sol"
```
